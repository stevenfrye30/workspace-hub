# Move checklist

Consolidated checklist. See topic files for details.

## 60+ days out
- [ ] Confirm move date & arrival address
- [ ] Research neighborhoods (see `orientation/neighborhoods.md`)
- [ ] Secure housing (lease or purchase) — `logistics/housing.md`
- [ ] Budget move (movers vs. DIY, storage)
- [ ] Notify current landlord / give notice
- [ ] Book movers / truck if using
- [ ] School enrollment (if applicable)

## 30 days out
- [ ] File USPS change of address
- [ ] Set up utility transfers (see `utilities.md`)
- [ ] Transfer / set up renter's or homeowner's insurance
- [ ] Find new primary care physician (`healthcare.md`)
- [ ] Research WI driver's license + car registration (`dmv-and-id.md`)
- [ ] Update driver's license/auto insurance provider on move date
- [ ] Notify bank, credit cards, employers, subscriptions

## Move week
- [ ] Pack essentials box (meds, chargers, documents, clothes for 3 days)
- [ ] Confirm movers 48 hr prior
- [ ] Final utility readings at old place
- [ ] Forward mail effective move date

## First 30 days in Milwaukee
- [ ] WI driver's license (required within 60 days of becoming a resident)
- [ ] WI vehicle title + registration (required within 60 days)
- [ ] Register to vote (WI allows same-day at polls; still update)
- [ ] Find primary care + dentist
- [ ] Join a neighborhood Facebook group / Nextdoor
- [ ] Try 3 local spots from `engagement/places-to-try.md`
- [ ] Walk around new neighborhood; meet neighbors

## First 6 months
- [ ] Establish routine (gym, coffee shop, grocery)
- [ ] Pick sports teams to follow
- [ ] Find a barber/stylist, mechanic, dentist
- [ ] Build small social circle (`engagement/community.md`)
