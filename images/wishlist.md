# World Gallery — Wishlist

Candidates for future batches. Grouped by region, then by sub-tradition, period,
or artist. When an item says "— Artist Name", that's a named creator and should
get the `artist` field in JSON (seeds the future artist view).

Items already in the gallery are marked `(HAVE)`. Delete or ignore those when
batching.

Workflow: pick ~5-10 from a region, use the Wikipedia API lookup pattern to
grab image URLs, paste into `add_works.py`, run `python add_works.py && python
build.py`.

## Suggested process

1. **Prioritize per batch.** Pick 5-10 per region, preferring items that
   (a) fill a gap you notice, (b) have clean Wikipedia pages, and (c) mix media
   (painting / sculpture / architecture / textile).
2. **Favor named artists** for Europe, Chinese/Japanese painting, and contemporary
   African/Americas. Add the `artist` field to seed the future by-artist view.
3. **Skip items that return bad thumbnails** (maps, photos of modern people,
   nothing). Note them, try a more specific page, or substitute.
4. **Keep descriptions one to two sentences.** Concrete detail beats survey.
